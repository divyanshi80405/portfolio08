## Running Locally

Clone the repository

```bash
git clone https://github.com/divyanshi80405/portfolio08.git
```

Navigate to the project

```bash
cd portfolio08
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```

Open

```
http://localhost:3000
```
